# Privacy Policy

Last updated: May 8, 2026

GocTruyenTranhVui Follow Exporter is a browser extension made by DZ0Phong.

## Single purpose

The extension exports the user's followed manga list and reading progress from `goctruyentranhvui23.com/truyen/theo-doi` to a local CSV file.

## Data accessed

The extension may read:

- Manga titles shown on the followed list page.
- Manga detail page URLs on `goctruyentranhvui23.com`.
- Visible reading progress shown by the website, such as `Đọc Tiếp #99`.
- Visible chapter metadata, such as latest chapter and read/unread state.

## Data collection and sharing

The extension does not collect, sell, transmit, upload, or share user data with any external server controlled by the developer.

All processing happens locally in the user's browser. The generated CSV file is created locally and downloaded by the browser.

## Permissions

The extension requests access to `goctruyentranhvui23.com` so it can read the followed list page and manga detail pages needed for export.

The extension uses the `tabs` permission only to check that the active tab is on the supported website before starting the export.

## Third-party services

The extension does not use analytics, advertising SDKs, remote tracking, or external APIs controlled by the developer.

## Contact

For issues or questions, please open an issue in the GitHub repository.
